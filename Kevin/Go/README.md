# Arreglos y Matrices — Go

Implementación de arreglos y matrices en Go.

## Contenido
```
Go/
├── Main.go       # Punto de entrada
├── arreglos.go   # Implementación de arreglos
├── matriz.go     # Implementación de matrices
└── go.mod        # Módulo Go
```

## Requisitos

- Go 1.21+

## Como ejecutar

Desde la carpeta `Go/`:
```bash
go run .
```

O para compilar un ejecutable:
```bash
go build -o arreglos
./arreglos
```

## Temas cubiertos

- Recorrido con `for` y `range`
- Modificación de elementos
- Búsqueda de valores
- Operaciones con matrices (Sumas)
