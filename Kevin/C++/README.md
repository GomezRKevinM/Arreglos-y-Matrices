# Arreglos y Matrices — C++

Implementación de arreglos y matrices en C++ moderno (C++20).

## Contenido
```
C++/
├── Main.cpp        # Punto de entrada
├── Arreglos.cpp    # Implementación de arreglos
├── Arreglos.h      # Declaraciones de arreglos
├── Matrices.cpp    # Implementación de matrices
├── Matrices.h      # Declaraciones de matrices
└── NumberUtils.h   # Utilidades numéricas
```

## Requisitos

- GCC 14+ (vía MSYS2 en Windows)
- CMake 3.20+
- CLion o cualquier IDE con soporte CMake

## Como ejecutar

1. Abre la carpeta raíz del proyecto en CLion
2. CLion detectará el `CMakeLists.txt` automáticamente
3. Selecciona el target `Arreglos_y_Matrices` en el dropdown
4. Presiona **Shift+F10** o el botón de play

## Temas cubiertos

- Recorrido con `for` y `forEach`
- Modificación de elementos
- Búsqueda de valores
- Operaciones con matrices (Sumas)
